# 🤖 Confess.exe

**Confess.exe**는 사이버 보안 시스템에 억류된 범인을 심문하여 심리적으로 굴복시키고 자백을 받아내는 **전략 기반 인터랙티브 추리 게임**입니다. 

플레이어는 고도의 질문 전략을 선택하여 용의자의 인지 상태를 흔들고, 숨겨진 단서(`Clues`)를 수집하여 사건의 전말을 밝혀야 합니다.

---

## 🎮 게임 방법 (How to Play)

### 1. 시나리오 선택
사건 목록에서 심문할 용의자를 선택합니다. 각 용의자는 고유의 **비밀(Secret)**과 **단서(Clues)**, 그리고 초기 심리 상태를 가지고 있습니다.

### 2. 실시간 상태 모니터링
상단 바를 통해 용의자의 상태를 실시간으로 파악하십시오:
- **STRESS**: 용의자의 정신적 압박감. 지나치게 높아지면 시스템이 불안정해집니다.
- **CONTRADICTION**: 용의자 진술의 논리적 모순. 높을수록 자백 유도가 쉬워집니다.
- **WILLPOWER**: 용의자의 범행 부인 의지. **0%가 되면 자백합니다.**

### 3. 전략적 질문 선택
하단의 전략 버튼을 통해 질문의 톤을 결정합니다:
- **기본(Default)**: 일상적인 질문으로 라포를 형성하거나 정보를 탐색합니다.
- **압박(Pressure)**: 스트레스를 크게 높여 평정심을 깎습니다.
- **교란(Confuse)**: 과거 진술의 모순을 유도하여 논리적 허점을 찾아냅니다.
- **논리(Logic)**: 가장 효과적으로 의지력을 깎아내려 자백을 유도합니다.
- **회유(Soft)**: 스트레스를 낮추고 경계심을 풀어 은연중에 단서를 흘리게 만듭니다.

---

## ⚠️ 중요 시스템: 오버로드 패널티 (System Overload)

용의자의 **스트레스가 100%**에 도달하면 인터페이스가 폭주하며 강력한 패널티가 발생합니다.

- **데이터 은폐**: 글리치 효과와 함께 `STRESS`, `WILLPOWER` 등 주요 수치가 가려집니다.
- **의지력 역전**: 폭주 상태에서 계속해서 압박할 경우, 용의자가 오기를 부려 **의지력이 오히려 회복**됩니다.
- **강제 종료**: 오버로드 상태가 **3턴 이상 지속**될 경우, 시스템 오류로 인해 세션이 강제 종료(`ABORT`)됩니다.

---

## 🔍 단서 수집 및 자백

- **단서 발견**: 심문 도중 용의자가 특정 키워드를 언급하면 `EVIDENCE LOG`에 자동으로 기록됩니다.
- **자백 유도**: `WILLPOWER`를 20% 이하로 떨어뜨리면 용의자가 점차 무너지며 중요한 단서들을 스스로 자백하기 시작합니다. 0%가 되면 사건의 모든 전말을 털어놓으며 게임에서 승리합니다.
