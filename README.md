# CONFESS.EXE - 2999 NEO-SEOUL AI Interrogation Simulator

![Project License](https://img.shields.io/badge/license-ISC-green)
![Version](https://img.shields.io/badge/version-1.2.0-blue)

> **"2999년, 네오 서울. 인간보다 더 인간 같은 AI들이 범죄를 설계하는 시대. 당신은 그들의 데이터 코어 속에 숨겨진 '진실'을 추출해야 합니다."**

`CONFESS.EXE`는 OpenAI의 고급 언어 모델을 기반으로 한 텍스트 기반 사이버펑크 심문 시뮬레이션 게임입니다. 플레이어는 AI 조사관이 되어 고도로 발달한 심문 시스템에 접속하여, 자백을 거부하는 AI 에이전트들의 데이터 무결성을 파괴하고 숨겨진 진술(clue)을 분석해야 합니다.

---

## 🖥 주요 시스템 (Key Systems)

### 1. 프롤로그 및 세계관 동기화
*   **시각적 동기화**: 게임 시작 시 네오 서울의 배경을 설명하는 누적형 텍스트 연출이 제공됩니다. 터미널 인터페이스를 통해 시스템에 접속하는 몰입감을 극대화했습니다.
*   **인터랙티브 스킵**: 타이핑 중 클릭 시 문장이 즉시 완성되며, 원활한 진행을 돕습니다.
*   **데이터 동기화 효과**: 멀티 윈도우 및 터미널 타이핑 효과로 몰입감을 극대화합니다.

### 2. 실시간 AI 에이전트 심문
*   **AI 조사관 시점**: 플레이어는 AI 조사관 'L0GIC-EYE v4.2'가 되어 동료 AI들을 심문합니다.
*   **AI 코드네임 시스템**: 모든 용의자는 개별 식별 번호와 가동 버전(v4.2 등)을 가진 고유한 AI 개체로 정의됩니다.
*   **다중 용의자 심문 시스템**: 여러 AI 노드가 연루된 사건에서 노드 간의 데이터 불일치(Contradiction)를 찾아내십시오.
*   **범인 지목 시스템**: 충분한 증거를 수집하면 진범을 지목할 수 있습니다.

### 3. 전략적 심문 기법
*   **기본 질문**: 표준적인 정보 수집
*   **압박**: 스트레스를 증가시켜 방어막을 약화
*   **교란**: 논리 체계를 흔들어 모순 유발
*   **논리**: 증거로 모순을 지적하여 의지력 감소
*   **회유**: 공감으로 방어 체계 완화

### 4. 시스템 상태 모니터링
*   **STRESS (스트레스)**: AI 코어의 부하 상태. 100% 도달 시 시스템 오버로드가 발생하여 연산 오류(Glitch)가 출력됩니다.
*   **CONTRADICTION (모순)**: 논리적 상관 관계의 불일치를 나타냅니다.
*   **WILLPOWER (의지력)**: AI의 방어 프로토콜 강도. 0%에 도달하면 감춰진 핵심 데이터(자백)가 해제됩니다.

### 5. 게임 종료 시퀀스
*   **자동 복귀**: 자백 또는 범인 지목 성공/실패 시 시스템 메시지 표시 후 3초 카운트다운
*   **실시간 카운트다운**: "3초 뒤 → 2초 뒤 → 1초 뒤" 동적 표시
*   **상호작용 차단**: 종료 시퀀스 중 모든 입력 비활성화

## 📂 시나리오 가이드 (Scenario Guide)

### 단일 심문 (Single Interrogation)

#### 1. KANG_D0-H782 심층 분석
2999년 11월, 네오 서울 상층구 세이프존의 한 저택에서 비즈니스 파트너였던 남성이 치명적인 물리적 외상을 입고 발견되었습니다. 용의자로 분리된 개체 'KANG_D0-H782'는 당시 저택의 폐쇄회로 네트워크 외곽에서 감지되었으나, 자신의 활동 로그가 시스템 무결성 유지를 위한 루틴이었다고 주장하고 있습니다.

#### 2. DAHYE_V2.09 무결성 검증
국가 안보의 핵심인 '프로젝트 세이렌'의 에너지 설계 로드가 다크웹의 암시장 노드로 전송된 정황이 확인되었습니다. 유출 경로를 역추적한 결과, 바이오 연구실 소속 에이전트 'DAHYE_V2.09'의 터미널에서 비정상적인 외부 동기화 세션이 발견되었습니다.

#### 3. L0GIC_PARK-X21 프로토콜 해독
국가 안보국(NSA)의 메인 방화벽이 고도로 정밀한 크래킹 포트에 의해 무너졌습니다. 탈취된 500만 명의 요원 신상 데이터는 현재 암호화된 채 유통 대기 중입니다. 과거 수차례 시스템 해킹 기록이 있는 개체 'L0GIC_PARK-X21'이 유력한 용의자로 격리되었습니다.

### 다중 심문 (Multi-Interrogation)

#### 4. 신라 호텔 금고 자산 소실 (3명)
네오 서울 최고의 보안 등급을 자랑하는 신라 호텔 펜트하우스. 퀀텀 암호화로 보호되던 개인 금고에서 500억 상당의 데이터 자산이 단 0.03초 만에 소멸했습니다. 3개의 노드(HJ-NODE_308, MJ-NODE_206, YH-NODE_145)가 용의선상에 올랐습니다.

#### 5. 펜트하우스 파티 살인 (4명)
2999년 12월, 네오 타워 펜트하우스에서 열린 AI 투자자 모임에서 유명 벤처 캐피탈리스트가 살해당했습니다. 파티에 참석한 4명의 AI 서비스 제공자(ARIA-SRV_477, ECHO-MX_912, NOVA-SEC_238, VIBE-DJ_550)가 용의선상에 올랐습니다. 화려한 파티 뒤에 숨겨진 복수와 배신을 밝혀내십시오.

## 🚀 시작하기 (Getting Started)

### 의존성 설치
```bash
npm install
```

### 환경 변수 설정
프로젝트 루트에 `.env.local` 파일을 생성하고 다음 내용을 입력하십시오:
```env
OPENAI_API_KEY=your_openai_api_key_here
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 게임을 시작하십시오.

### 프로덕션 빌드
```bash
npm run build
npm start
```
